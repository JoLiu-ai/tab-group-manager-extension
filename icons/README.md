# 图标文件说明

本文件夹应包含以下三个图标文件：

- `icon16.png` - 16x16 像素，用于扩展列表
- `icon48.png` - 48x48 像素，用于扩展管理页面
- `icon128.png` - 128x128 像素，用于 Chrome 网上应用店

## 如何获取图标

### 方法一：使用在线工具
1. 访问 https://www.favicon-generator.org/
2. 上传你的图片或使用在线设计工具
3. 下载不同尺寸的图标

### 方法二：使用图片编辑软件
使用 Photoshop、GIMP 或其他图片编辑软件创建图标

### 方法三：使用占位图标
如果暂时没有图标，可以：
1. 创建一个简单的纯色 PNG 图片
2. 使用任何图片，重命名为对应的文件名
3. 扩展功能不受影响，只是图标显示可能不美观

## 图标设计建议

- 使用简洁的设计，在小尺寸下也能清晰识别
- 建议使用与"分组"、"文件夹"相关的图标
- 颜色建议使用蓝色系，与 Chrome 风格一致
- 确保图标在不同背景下都清晰可见
